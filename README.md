# ⌨️ KeyCheck

### Testador de Teclado e Mouse para Técnicos de Suporte e Infraestrutura

Ferramenta web para validar rapidamente o funcionamento de teclados e mouses durante manutenção, suporte técnico, montagem de equipamentos e diagnóstico de hardware.

## Sobre

Durante minha rotina como **Analista de Infraestrutura**, frequentemente preciso validar teclados e mouses antes de entregar equipamentos aos usuários.

Embora existam diversos sites para esse tipo de teste, decidi desenvolver uma ferramenta própria, com foco em:

- Interface moderna
- Facilidade de uso
- Compatibilidade com diferentes layouts
- Código limpo
- Possibilidade de evolução futura

---

# Funcionalidades

- Teste completo do teclado
- Layout ABNT2 (Brasil)
- Layout ANSI (EUA)
- Alternância entre layouts
- Destaque visual das teclas
- Teste dos botões do mouse
- Histórico de eventos
- Contador de teclas testadas
- Exibição da última tecla pressionada
- Exibição do código da tecla (`KeyboardEvent.code`)
- Interface responsiva
- Funciona diretamente no navegador
- Não necessita instalação

# Como executar

Clone o projeto

```bash
git clone https://github.com/SEU-USUARIO/keycheck.git
```

Entre na pasta

```bash
cd keycheck
```

Abra o arquivo

```
index.html
```

Ou utilize a extensão **Live Server** do VS Code.
