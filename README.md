# KeyCheck — Testador de Teclado e Mouse

Projeto web para testar teclas do teclado e botões do mouse diretamente pelo navegador.

## Funcionalidades

- Layout principal ABNT2 Brasil.
- Alternância entre BR e US.
- Teclado visual alinhado em grade fixa.
- Teclas cinza quando não acionadas.
- Teclas vermelhas enquanto pressionadas.
- Teclas verdes após liberadas.
- Teste de botões do mouse.
- Histórico dos últimos eventos.
- HTML, CSS e JavaScript puro.

## Como usar

Abra o arquivo `index.html` no navegador.

## Como rodar no VS Code

1. Abra a pasta no VS Code.
2. Instale a extensão Live Server.
3. Clique com o botão direito no `index.html`.
4. Clique em `Open with Live Server`.
